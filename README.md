# DeepHUD table
The DeepHUD data is published in the `/SmartDashboard/DeepHUD` table

| Key | contains |
| -- | -- |
| match_time | The time reported from FMS |
| speed | Speed that is passed into ArcadeDrive |
| mode | 0 = auto, 1 = teleop |
